## Implemented behaviour

Added deterministic internal analyst tools over dashboard data:

- `get_dashboard_summary(data)`
- `get_top_posts(data, limit=3)`
- `get_underperforming_posts(data, limit=3)`
- `get_repeat_candidates(data)`
- `get_pause_candidates(data)`
- `get_retention_issues(data)`
- `compare_posts_by_metric(data, metric, limit=3)`

These tools accept the existing `dashboard_data.json` structure, return safe
structured fields only, avoid mutation, avoid network calls, and handle missing
optional fields gracefully.

Manual analyst intent routing now uses these tools for:

- best/winner/performed/top/strongest questions
- hook/reuse/repeat/post tomorrow questions
- pause/avoid/weak/worst questions
- retention/watch/drop/hold/pacing questions
- underperforming/stuck/90 views/low views questions
- general summary questions

Manual analyst responses now include an optional `trace` object with:

- `interpreted_intent`
- `tools_used`
- short factual `observations`
- `limitations`

The trace is a transparent tool-use summary, not hidden chain-of-thought. It is
optional in response validation, so older provider responses without trace
remain valid.

The frontend renders a collapsible "How this answer was grounded" section only
when `trace` is present. Older responses without trace continue to display.

## Known issues and limits

- OpenAI and Claude analyst providers still accept optional trace in validation,
  but the deterministic tool trace is implemented for manual analyst mode only.
- The trace records tool-use summaries and observations only; it deliberately
  does not expose chain-of-thought.
- The analyst remains grounded only in `outputs/latest/dashboard_data.json`.
  It cannot diagnose impressions, traffic source, follower status, current
  trend/audio metadata, or TikTok distribution behaviour.
- No frontend test framework was added because the project does not currently
  have frontend tests.
- The full test run still emits the existing FastAPI/Starlette test-client
  deprecation warning; tests pass.

## Deliberately not implemented

- no SaaS features
- no auth, database, billing, tenancy, Stripe, deployment, or background jobs
- no new analyst endpoint
- no dashboard data schema change
- no frontend API keys
- no duplicated OpenAI, Claude, Airtable, or TikTok clients
- no live OpenAI or Claude calls in tests
- no hidden chain-of-thought
